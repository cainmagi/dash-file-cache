```python
DownloaderURL(
    url: str,
    file_name_fallback: str | None
)
```

`DownloaderURL` is a `TypedDict`.

`DownloaderURL` is the component property `Downloader(url=...)`. It defines the URL
used to access the data to be downloaded.

Each time when this value is set, a download event will be triggered. After
triggering the download event, this value will be reset by a blank string.
