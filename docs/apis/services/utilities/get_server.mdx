```python
server: flask.Flask = get_server(app: dash.Dash | flask.Flask)
```

Get the Flask server instance from the application.

## Arguments

### Requires

| Argument |            Type            |             Required              | <center>Description</center>                                                                                                                |
| :------: | :------------------------: | :-------------------------------: | ------------------------------------------------------------------------------------------------------------------------------------------- |
|  `app`   | `dash.Dash \| flask.Flask` | <InlineIcon icon={vsiCheckIcon}/> | Can be either a dash or a flask app. If it is a dash app, try to fetch the flask server instance. Otherwise, return the flask app directly. |

### Returns

| Argument |     Type      | <center>Description</center>                  |
| :------: | :-----------: | :-------------------------------------------- |
| `server` | `flask.Flask` | The Flask server instance of the given `app`. |
