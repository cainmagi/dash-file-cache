# Overview of APIs

<APITopBar type="package" source="." />

Utilities for providing convenient methods to serve cached data in Plotly-Dash or Flask.

This graph shows the high-level overview of the whole package. The most commonly used
functionalities are the caches like [`CachePlain`](./apis/caches/memory/CachePlain.mdx),
the service factory [`ServiceData`](./apis/services/data/ServiceData.mdx),
and the component [`Downloader`](./apis/components/downloader/Downloader.mdx).
