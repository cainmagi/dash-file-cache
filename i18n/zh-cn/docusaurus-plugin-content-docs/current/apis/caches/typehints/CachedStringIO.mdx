## 关键字 {#keywords}

| 关键字 |   类型    | <center>说明</center>                                         |
| :----: | :-------: | :------------------------------------------------------------ |
| `type` |  `"str"`  | 缓存数据的类型。                                              |
| `data` | `IO[str]` | 缓存的文字数据。该文件的类型可以透过文件名和`mime_type`决定。 |
