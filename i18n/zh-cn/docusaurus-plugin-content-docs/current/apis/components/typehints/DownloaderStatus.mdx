```python
DownloaderStatus(
    type: "success" | "error-connect" | "error-config" | "error-io" | "error-unknown",
    http_code: int
)
```

`DownloaderStatus`是`TypedDict`。

`DownloaderStatus`是`Downloader(status=...)`的属性类型。其定义了下载事件结束时设置的状态码。

若同一下载器触发了多次下载事件，后触发的事件将会覆写先触发的事件设置的状态。

## 关键字 {#keywords}

|   关键字    |                                       类型                                        | <center>说明</center>                                                   |
| :---------: | :-------------------------------------------------------------------------------: | :---------------------------------------------------------------------- |
|   `type`    | `"success" \| "error-connect" \| "error-config" \| "error-io" \| "error-unknown"` | 事件的状态码。若下载事件成功，则下载事件结束时、该值当设为`"success"`。 |
| `http_code` |                                       `int`                                       | 响应的HTTP状态码。若下载事件成功，则该值应当处于`200-299`的范围内。     |
