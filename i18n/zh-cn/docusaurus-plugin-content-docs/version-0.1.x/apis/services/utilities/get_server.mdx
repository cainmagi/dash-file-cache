```python
server: flask.Flask = get_server(app: dash.Dash | flask.Flask)
```

从一个应用获取Flask服务应用的实例。

## 参数 {#arguments}

### 输入 {#requires}

| 参数  |            类型            |               必选                | <center>说明</center>                                                                                 |
| :---: | :------------------------: | :-------------------------------: | ----------------------------------------------------------------------------------------------------- |
| `app` | `dash.Dash \| flask.Flask` | <InlineIcon icon={vsiCheckIcon}/> | 可以是一个Dash或Flask应用。若为Dash应用，尝试获取其Flask服务实例。否则，直接返回Flask应用。 |

### 输出 {#returns}

|   参数   |     类型      | <center>说明</center>            |
| :------: | :-----------: | :------------------------------- |
| `server` | `flask.Flask` | 给定`app`的Flask服务应用实例。 |
