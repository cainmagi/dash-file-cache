`CachedBytesIO`是`TypedDict`。

`CachedBytesIO`是`BytesIO`数据的缓存格式。

## 关键字 {#keywords}

| 关键字 |    类型     | <center>说明</center>                                           |
| :----: | :---------: | :-------------------------------------------------------------- |
| `type` |  `"bytes"`  | 缓存数据的类型。                                                |
| `data` | `IO[bytes]` | 缓存的二进制数据。该文件的类型可以透过文件名和`mime_type`决定。 |
